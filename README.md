## 简介
本仓库是Up70电控组的软件驱动库，包含了许多驱动和标准文件。

## 注意事项
部分文件在编写之初并没有考虑 ***移植性*** 和 语言风格的 ***一致性*** 问题；同时，有部分文件，来源于前两个赛季的工程导出后修改，因此代码可能经由未知的不同作者加工。

代码规范性的问题将在之后渐进解决。

关于代码规范的问题，请查看：[Up70电控代码规范](https://www.atstarcorner.net/coderule/)
